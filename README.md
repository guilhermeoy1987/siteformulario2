# Projeto de Formulário Estruturado - EBAC

Este repositório contém o desenvolvimento de um formulário web focado em organização de dados e usabilidade (UX).

## O que foi realizado:
* **Estruturação de Dados:** Criação de seções organizadas para coleta de diferentes tipos de informações.
* **Acessibilidade e Semântica:** Uso de tags HTML5 para garantir que o formulário seja acessível e bem interpretado pelos navegadores.
* **Estilização Customizada:** Aplicação de CSS para melhorar o visual dos campos e a experiência de preenchimento.
* **Validação de Campos:** Uso de atributos nativos para garantir que os dados essenciais sejam inseridos pelo usuário.

## Tecnologias:
* HTML5
* CSS3
* Git / GitHub
